🐘 ШПАРГАЛКА №4: PostgreSQL

Запуск, подключение, бэкап и восстановление

Запуск PostgreSQL

docker run -d --name postgres-lab -e POSTGRES_USER=devops -e POSTGRES_PASSWORD=devops123 -e POSTGRES_DB=learning -p 5432:5432 -v postgres_data:/var/lib/postgresql/data postgres:15

Подключение к базе

docker exec -it postgres-lab psql -U devops -d learning

Команды внутри psql

\\l # список всех баз данных
\\dt # список таблиц
\\d employees # структура таблицы
SELECT * FROM employees;
\\q # выход

Бэкап и восстановление

docker exec postgres-lab pg_dump -U devops learning > backup.sql
cat backup.sql | docker exec -i postgres-lab psql -U devops -d learning
