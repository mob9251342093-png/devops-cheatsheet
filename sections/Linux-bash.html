Навигация и файлы

ls -la # показать все файлы (включая скрытые)
cd /home/devops # перейти в папку
pwd # показать текущую папку
cat file.txt # посмотреть содержимое файла
tail -f /var/log/app.log # следить за логом в реальном времени

Поиск и анализ

grep "error" app.log # найти строки с текстом
grep -i "error" app.log # поиск без учёта регистра
grep -v "info" app.log # все строки, КРОМЕ info
wc -l app.log # количество строк в файле

Управление системой

chmod +x script.sh && ./script.sh # сделать скрипт исполняемым и запустить
df -h # свободное место на диске
sudo apt update && sudo apt upgrade -y # обновить систему
systemctl status docker # статус сервиса
journalctl -u docker -f # логи сервиса в реальном времени
