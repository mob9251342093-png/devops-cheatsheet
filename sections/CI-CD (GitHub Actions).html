🔄 ШПАРГАЛКА №5: CI/CD (GitHub Actions)

Автоматическая проверка кода при каждом push

Базовый workflow для проверки bash-скриптов

name: Check Bash Scripts on: push jobs: build: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 - run: bash -n *.sh

Что означают ключевые слова

Ключевое словоЧто делает
nameНазвание workflow (видно в GitHub Actions)
on: pushЗапускается при каждом push
jobsСписок задач для выполнения
runs-onНа какой системе запускать (ubuntu, windows, macos)
stepsКонкретные шаги: скачать код, запустить проверку
actions/checkout@v4Скачивает код из репозитория
💡 Главное правило: CI/CD ловит ошибки на раннем этапе. Если workflow упал — исправляй сразу.
